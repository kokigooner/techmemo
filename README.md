# Tech-dictionary

"Tech-dicyionary"は簡潔に言えば単語帳なようなものである。
覚えたい単語を登録して一覧で見ることができる。
また、トップページでは意味は隠されているので使い方次第では
一問一答のように使うことができる。
開発に至った経緯は単純でプログラミングを学習していて
このようなサイトがあったらいいなと思ったからである。
学んだことがしっかり身についているか確認をする為に作成した。
